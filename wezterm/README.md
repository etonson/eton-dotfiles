# WezTerm Configuration (Pro Edition)

這是由 Alacritty Pro 遷移並深度優化的 WezTerm 設定檔，整合了 Ubuntu 桌面環境與多項開發效率工具。

## 📋 進階特色

- **配色方案**: 經典 **Dracula** 主題（精確移植）。
- **視覺特效**: 85% 透明度 + 背景模糊，移除冗餘標題列以實現極簡風格。
- **字體優化**: **JetBrainsMono Nerd Font** (14px)，行高 `1.15` 提升閱讀舒適度。
- **URL 智慧偵測**: 支援 `ipfs`, `git`, `ssh`, `magnet` 等多種協定。
- **Ubuntu 整合**: 支援檔案管理員 (Nautilus) 右鍵「在 WezTerm 開啟」。
- **版本控管**: 設定檔位於 `~/.config/wezterm/`，方便 Git 管理。

## ⌨️ 快捷鍵清單 (Key Bindings)

| 動作 | 快捷鍵 |
| :--- | :--- |
| **複製 / 貼上** | `Ctrl + Shift + C` / `V` (支援選取即複製) |
| **搜尋內容** | `Ctrl + Shift + F` |
| **網址跳轉 (Hints)** | `Ctrl + Shift + U` (進入 QuickSelect 模式) |
| **進入 Vi 模式** | `Ctrl + Shift + Space` (Activate Copy Mode) |
| **字體放大 / 縮小** | `Ctrl + +` / `-` |
| **重置字體大小** | `Ctrl + 0` |
| **全螢幕切換** | `Alt + Shift + Enter` |

## 🛠️ 安裝與部署

### 1. 必要條件
- **WezTerm**
- **JetBrainsMono Nerd Font**
- **zsh** & **python3-nautilus** (用於右鍵選單)

### 2. 套用設定
將此儲存庫複製到 `~/.config/wezterm`：
```bash
git clone git@github.com:etonson/eton-wezterm.git ~/.config/wezterm
```

### 3. 啟用 Shell 整合
將以下內容加入 `~/.zshrc`：
```bash
[[ -f ~/.config/wezterm/wezterm.lua ]] && eval "$(wezterm shell-integration --posix)"
```
